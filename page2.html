Attendance Analysis for Non-Profit Assistance Program

Project Objectives

This project involves analyzing the attendance of participation in a non-profit project that assists low-income individuals with administrative tasks, job searches and other public affairs. The goal is to provide valuable insights that can help improve the organization's services.

Details to achieving project objectives

  • Analyzing the attendance and retention rates of participation for the non-profit project
  • Identifying one-time attendees versus returning participants
  • Grouping the data by day of the week to count the number of attendees on each day
  • Extracting the day of the week from the "Date" column in Excel using Power Query
  • Creating an interactive report using Power BI to summarize the insights gained from the data analysis

Project execution steps

  1. Data Entry: Attendance data was manually entered into the "List" workbook using Exce.
  2. Data Cleaning: The data was cleaned in Excel by checking names for errors or typos and removing irrelevant data.
  3. Transformation: Power Query was used to transform the data by extracting the day of the week from the "Date" column and identifying one-time attendees versus returning ones.
  4. Data Analysis: DAX was used in Power BI to create measures that calculated the total number of attendees, total number of days attended, number of one-time attendees, number of returning attendees, percentage of one-time attendees and returning attendees.
  5. Data Visualization: Power BI was used to create visualizations, including a bar chart to show the total number of visits by day of the week, a column chart to show total number of visits by month, a treemap chart to show the number of one-time attendees versus returning attendees, a donut chart to show the participant attendance since the project started until present day.
  6. Data Presentation: An interactive report was created using Power BI to summarize the insights gained from the data analysis. The report included visualizations and a narrative to explain the patterns observed in the data. Drill-down functionality was used to make the report more interactive and informative.

Conclusion and Recommendations

Based on the analysis of the attendance and retention rates for the non-profit project, it is recommended that the organization allocate its resources according to the busiest months and days of the week. By doing so, they can ensure that they are meeting the demand and providing assistance at highest quality.

Furthermore, it is recommended that the organization gathers a wider scope of information for future data analysis. This could include collecting demographic information about the clients, such as age, gender, and income level, to gain a better understanding of their needs and preferences. This additional data can be used to further refine the organization's services and improve the effectiveness of their programs.

In conclusion, this project has provided valuable insights into the attendance and retention rates of participants for the non-profit project. By utilizing Excel, Power Query, and Power BI, I was able to identify patterns and trends in the data that can inform decision-making and improve the organization's services. With the recommendations provided, the organization can continue to enhance their programs and better serve the needs for their project.
